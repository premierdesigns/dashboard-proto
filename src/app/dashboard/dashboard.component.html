#### FUNCTIONALITY

      Each "white box" on the dashboard page represents a card with a widget showing data.
      Some of these boxes will be customizable by the jeweler, some will not.

      ##### Some box ideas

      - Sales
        - Year-to-date
        - vs. previous year
        - past six months (or some other amount)
      - Upcoming Shows
      - Current promotions
      - New jewelers in my downline

      #### DATA

      The data that shows up and needs to be connected here will vary and change based on what cards/widgets are showing on the dashboard.
