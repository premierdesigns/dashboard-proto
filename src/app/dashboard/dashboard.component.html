#### FUNCTIONALITY

      - Each "white box" on the dashboard page represents a card with a widget showing various bits of data
      - Some of these boxes will be customizable by the jeweler, some will not.
      - Action bar across the top will be for search, help/support, quick-add (order, show, contact, etc), and notifications
      - Cards can be customized on where they sit and how big they are
